# Con Chim Nau Chin

## Installation

1. cp .env.example .env
4. composer install
5. connect DB with info env
6. php artisan key:generate
7. php artisan migrate --seed

## Install node_modules

    yarn or npm install

## Run scripts when files change

    yarn watch or npm run watch

## Format .js, .scss

    yarn format

## Open web

**http://localhost:8080**
